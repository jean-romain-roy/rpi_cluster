RPI Setup

1. SSH in Rpi

2. Edit Hostname, Password and Network location with : sudo raspi-config

3. Reboot/Re-ssh : sudo reboot

4. Update rpi os with : sudo rpi-update

5. Reboot/Re-ssh : sudo reboot

6. pull information about the latest versions of packages available : sudo apt-get update

7. upgrade the installed software to its latest versions : sudo apt-get upgrade
