RPI Setup

1. SSH in Rpi

2. Edit Hostname and Network location with : sudo raspi-config

3. Update rpi os with : sudo rpi-update

4. Reboot : sudo reboot

5. Generate safe passwordless login files by using (keep pressing enter to skip options) : ssh-keygen -t rsa

6. pull information about the latest versions of packages available : sudo apt-get update

7. upgrade the installed software to its latest versions : sudo apt-get upgrade


Optional :

8. install packages : 
	sudo apt-get install sysbench
	sudo apt-get install nmap


9. to run sysbench write : sysbench --test=cpu --cpu-max-prime=2000 --num-threads=2 run
